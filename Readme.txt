Projet: Lafleur

Framework: Framework Pedagogique (auteur MERY Ludovic)

Auteur: MERY Ludovic


Cette archive contient un projet complet utilisé cette année (2013/2014) avec mes étudiants SLAM.

Il repose sur l'usage d'un framework minimaliste montrant l'usage:
du modèle MVC
d'un ORM

Ce projet est uniquement à but pédagogique.

Le repertoire db contient le script de la base de données MySQL.
Le repertoire config contient un fichier XML de configuration (accès a la base de données et un compte d'administration du back office)


La partie Vue repose sur le framework Twitter Bootstrap
